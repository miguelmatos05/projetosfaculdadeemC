📘 Projetos em C – FEUP (Engenharia Eletrotécnica)

Este repositório contém exercícios e projetos em linguagem C/C++ desenvolvidos no âmbito das cadeiras de programação da FEUP – Faculdade de Engenharia da Universidade do Porto, no curso de Engenharia Eletrotécnica e de Computadores.

🎯 Objetivo

O objetivo deste repositório é:

Consolidar conceitos fundamentais de programação em C/C++
Praticar estruturas de dados (vetores, listas, pilhas, filas)
Desenvolver pensamento algorítmico
Preparação para avaliações e exames
📂 Conteúdo do repositório

O repositório pode incluir:

✔ Exercícios de vetores e arrays
✔ Funções e modularização em C
✔ Algoritmos de pesquisa e padrões
✔ Trabalhos práticos da disciplina
🛠️ Tecnologias usadas
Linguagem C (C99 / C11)
GCC
Code::Blocks / VS Code / CLion (dependendo do ambiente)
🚀 Como compilar e executar
Linux / Mac
gcc ficheiro.c -o programa
./programa
